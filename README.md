## Description
**Blad** is a desktop Markdown editor built with **JavaFX**.
It provides a clean and focused writing environment with a preview mode, allowing users to write Markdown and quickly switch views to see the rendered output.

The goal of the project is to create a minimalistic and lightweight Markdown editor for simple tasks and fast reading.

## Features
  * Markdown preview
  * Clean and minimal user interface
  * File open and save support (.md files)
  * Keyboard-focused workflow

## Tech Stack
  * Java 21
  * JavaFX
  * Flexmark (MArkdown parsing and rendering)
  * Maven
> **Note:** JUnit is included in the project configuration but is not currently used for testing.

## Installation
### Prerequisites
  * Java SDK 21 or later
  * JavaFX (if not bundled with your JDK)

### Steps
```
git clone https://github.com/your-username/blad.git
cd blad
```
Open the project in your IDE (IntelliJ IDEA recommended) and run the main application class.
